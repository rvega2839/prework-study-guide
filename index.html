HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The language of the page is determined before the head element.
  • The 'p' tag is used create a paragraph in an ordered fashion
  • Metadata is information provided to the browser in order to render the page correctly.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: Applies to a specific tag to style a particular page element only.
  • Internal style sheet: contains rules that apply to all tagged page elements.
  • External CSS style sheet: It allows them to keep all the CSS rules in a seperate file.
  • External CSS style sheet is prefered by most developers because it makes design changes easier.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • We can use the "console.log" method to output a message to the web console by adding an argument.
  • the script.js file is linked within the body element.
